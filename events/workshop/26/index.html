Description

A blockchain is a growing list of records, called blocks, that are linked together using cryptography. Each block contains a cryptographic hash of the previous block, a timestamp, and transaction data. This workshop will cover:-
1. Introduction to Cryptocurrency
2. Current Trends in Cryptocurrency
3. Future of Cryptocurrency
4. Where to start
5. Limitations of Cryptocurrency
6. Career Opportunities
